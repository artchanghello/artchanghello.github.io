温阳
虚证治法
把体内衰退的阳气温热起来，恢复温煦功能。主用于阳虚证。
温 → 让冷变热
阳 → 脾阳/肾阳/全身阳气
画面：冬天屋里冷 -> 打开暖气片 -> 全屋逐渐暖起来。 辨："温"是缓慢升温（vs "回阳"是救极寒）。阳气虚但还没到快没的状态，用温。

辨证要点：畏寒肢冷/舌淡苔白/脉沉弱

回阳
虚证治法
让已经快要消失的阳气回来。是"救逆"中最紧急的情况——用于亡阳证（阳气暴脱）。
回 → 回来了（极寒开始暖）
阳 → 肾阳（根阳）
画面：锅炉快熄灭了（脉微欲绝）-> 大火猛烧 -> 烟囱重新冒烟（脉回来了）。 辨：比"温阳"程度重得多。温阳是调养，回阳是救命。

温中
虚证治法
温暖脾胃中焦。脾胃虚寒、完谷不化、腹痛喜温喜按时用。
温 → 温热
中 → 脾胃中焦
画面：脾胃是灶台上的锅 -> 锅底火太小（脾阳虚）-> 饭煮不熟（完谷不化）-> 加柴火（温中）。 辨："温中"≠"温阳"。"温阳"范围大（可指肾阳），"温中"专指脾胃。

温里 / 散寒
虚证治法
温里：温化脏腑内部的寒邪（内寒）。散寒：把在表的寒气驱散出去（外寒）。
温里 → 内部加温
散寒 → 推出去赶走
辨：温里 vs 散寒 = 里寒 vs 表寒。 散寒 = 麻黄/桂枝把寒推出去（门开，风雪吹走）。 温里 = 干姜/附子从里面加热（灶台生火，整个屋暖）。

辨证要点：内寒/脘腹冷痛

温肾阳 / 温下
虚证治法
温肾阳：温补肾中阳气，主治肾阳虚（水肿/畏寒/夜尿多/五更泻）。温下：温化下焦寒积（寒秘/寒凝少阴）。

温经发表
虚证治法
温通经脉 + 发散表邪，表里双解。主治太阳表证+少阴里寒同时存在（表里俱寒）。
温经 → 温通少阴里寒（附子走里）
发表 → 发散太阳表寒（麻黄走表）
画面：外有大雪封门（太阳表寒），内有地窖结冰（少阴里寒）-> 麻黄散表+附子温里，内外同时加热。 辨：表证明显→麻黄附子细辛汤(65)。里虚为主→先温里后解表（次序不能乱）。

益气 / 补气
虚证治法
补充人体的气，增强推动/固摄/防御/气化功能。主治气虚证（神疲乏力、少气懒言、脉弱）。
益 → 增加（锦上添花）
补 → 填补（雪中送炭）
画面：锅炉火太小（气虚）-> 火力不够 -> 加煤添柴（益气/补气）。 辨："益气"偏于增强，"补气"偏于填补亏损。常用药：黄芪/人参/党参。

补中 / 升提 / 升阳 / 举陷
虚证治法
补中：补脾胃中焦之气。升提/升阳：把下陷的气向上托举。举陷：把中气下陷的症状抬回来。
补中 → 把中焦气补满
升提 → 向上拉（↓变↑）
画面：脏器挂在绳子上，绳子松了 -> 脏器往下掉（气陷）-> 把绳子往上拽（升提）。 辨：补中≠升提。补中只是补满，升提是在补满的基础上往上拉。黄芪+柴胡+升麻是升提药对。

养血 / 补血
虚证治法
补充血液，濡养全身。主治血虚证（面白无华、头晕心悸、手足麻木、脉细）。
养 → 慢慢滋养（缓补）
补 → 直接填补（急补）
画面：井里水太少了（血虚）-> 慢慢往里注水（养血）。 辨：养血偏缓（慢性血虚），补血偏急（严重失血后）。当归/白芍/熟地是养血药。

滋阴 / 养阴 / 润燥
虚证治法
滋阴：填补真阴（肾阴/肺阴/肝阴）。养阴：滋养阴液（比滋阴温和）。润燥：滋润燥邪伤津导致的干燥。
滋 → 大补真阴（水库注满）
养 → 小补阴液（细水长流）
润 → 滋润干涩
辨：阴虚=水库干了（急需补水）-> 滋阴（大剂量）。燥邪=外面太干把水吸走了 -> 润燥（滋润+清热）。

建中 / 和中 / 补中
虚证治法
建中：重建中焦脾胃功能（在脾虚+里急时用）。和中：调和脾胃（脾胃不和/呕逆）。补中：填补中焦之气。
建中 → 重建（中焦功能失常才用）

清热
实证治法
清除体内热邪。根据热在气分/血分/三焦，选用不同层次的清热药。
清 → 清除干净

泻火
实证治法
用苦寒药直折火势，主清气分大热。高热、烦渴、汗出、脉洪大时用。
泻 → 强力压下去
火 → 熊熊烈火（气分热）
画面：森林大火 -> 投灭火弹（苦寒直折）-> 火势骤降。 辨："泻"字语气最重，适用于实火嚣张，必须压制的场合。

凉血
实证治法
清血分热邪。热在血分表现为：各种出血（衄血/便血/经血不止）、斑疹鲜红、血热妄行。

生津 / 止渴
实证治法
生津：生成津液，补充热邪伤及的阴液。止渴：解除口渴感（虚渴或实渴）。

清热燥湿 / 泻肝 / 清肝
实证治法
清热燥湿：用苦寒药清热的同时把湿干燥掉。泻肝：泻肝胆实火。清肝：清肝经郁热。

清虚热 / 滋阴清热
实证治法

利水 / 利湿 / 渗湿
实证治法
利水：使水饮从小便排出（用于水肿/痰饮）。渗湿：用甘淡之品渗利湿邪（比利水轻）。利湿：使湿邪从小便分消（介于利水与渗湿之间）。
利 → 打开闸门放水
渗 → 慢慢渗出
湿 → 湿邪分消
辨：水肿（沉重明显）-> 利水（强，茯苓/泽泻）。 湿重（酸胀为主）-> 渗湿（缓，薏苡仁/茯苓）。 利水常用：茯苓/泽泻/猪苓。渗湿常用：薏苡仁/茯苓。

化饮
实证治法
把水饮（液态/稠浊的水液）化开，变成可代谢的气态或从小便排出。主治痰饮病。
